GAMES

This page contains a number of recreations of pre-existing games which I have created throughout my Computer Science A-Level. Some of these games were programmed through in-lesson demonstrations and others were created in my own time using tutorials from YouTube such as The Coding Train. Taking the time to make these games has helped improve my programming skills, covering the use of two-dimensional arrays, iteration, classes, abstraction, etc. All of these games can be played by clicking on the image.
